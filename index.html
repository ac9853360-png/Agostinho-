Teste Qui-Quadrado de Associação

O que é?

O Teste Qui-Quadrado (χ²) é um teste não-paramétrico utilizado para determinar se existe uma associação significativa entre duas variáveis categóricas (ex: Fumar [Sim/Não] vs. Cancro Pulmonar [Sim/Não]).

Hipóteses

  • H0 (Hipótese Nula): As variáveis são independentes (não há associação).
  • H1 (Hipótese Alternativa): As variáveis são dependentes (existe associação).

Como funciona?

O teste compara as frequências observadas (os seus dados reais) com as frequências esperadas (o que esperaríamos se H0 fosse verdadeira). Se a diferença for grande, rejeitamos H0.

Insira a matriz de dados (frequências). Cada linha representa um grupo, separe os valores por vírgulas.

ANOVA (Análise de Variância)

O que é?

A ANOVA One-Way avalia se existem diferenças estatisticamente significativas entre as médias de três ou mais grupos independentes. (Para 2 grupos, usaríamos o Teste T-Student).

Hipóteses

  • H0: µ1 = µ2 = µ3 = ... (Todas as médias populacionais são iguais).
  • H1: Pelo menos uma média populacional é diferente das restantes.

O Racional (Estatística F)

A ANOVA divide a variância em duas partes: a variância entre os grupos (devido ao tratamento) e a variância dentro dos grupos (erro aleatório). Se a variância entre os grupos for muito maior, a Estatística F será alta, indicando que os tratamentos têm efeitos diferentes.

Insira os dados numéricos de cada grupo (separados por vírgula), uma linha por grupo (mínimo 3 grupos).

Correlação e Regressão Linear Simples

O que são?

Correlação de Pearson (r): Mede a força e a direção da relação linear entre duas variáveis contínuas. Varia de -1 (inversa perfeita) a 1 (direta perfeita).

Regressão Linear (y = mx + b): Vai um passo além. Permite prever o valor da variável dependente (Y) com base na variável independente (X).

Conceitos Chave

  • R² (Coeficiente de Determinação): Indica a proporção da variância em Y que é previsível a partir de X. Ex: R² = 0.80 significa que 80% da variação é explicada pelo modelo.

Insira as duas variáveis. (X: Independente/Dose, Y: Dependente/Efeito). Devem ter o mesmo número de elementos.
